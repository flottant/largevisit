ZY player#

Github 上的高星开源视频资源播放器。

文档：http://zyplayer.fun/doc/

最新版帮助：

安装新版本后不工作/部分功能不正常

数据文件夹位置可参考相应系统下chrome的配置文件夹路径

Windows下,请清理 AppData\Roaming\zy文件夹

linux下，请清理~/.config/zy

MacOS下，请清理Users/用户名/Library/Application Support/zy

MacOS更新报错，重新安装打不开A JavaScript error occurred in the main process请参考 #593

清理后重置软件，再重试.

安装新版后部分功能不正常也可以按上述操作，如果还是不行请报bug

以下摘自 github 官方介绍页面。

官网：http://zyplayer.fun/

参与开发：https://github.com/Hunlongyu/ZY-Player/

✨特性#

  • 🍕 全平台支持. Windows, Mac, Linux
  • 🍥 支持 IPTV, 卫视直播
  • 🍔 视频源支持自定义, 支持导入, 导出
  • 🍟 支持海报模式和列表模式浏览资源
  • 🌭 播放历史, 自动跳转历史进度
  • 🍿 收藏夹支持导入,导出, 支持同步追剧
  • 🥙 支持精简模式, 摸鱼划水
  • 🥪 一键搜索所有资源, 支持历史搜索记录
  • 🌮 导出资源下载链接
  • 🍣 支持第三方播放器播放
  • 🍤 显示豆瓣评分
