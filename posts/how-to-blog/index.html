给已经有经验的博客使用者。

为什么要写博客？#

当前网络环境下，个人博客能够尽可能地展现个性，避免各种扰人的监管。

静博还是动博？#

白嫖党就选静博吧。对技术力有较高要求。

推荐结合方式：

hexo + github pages + vercel + github actions

主题：yun/volantis/butterfly/fluid/next/suka/cards/paper 等

利用github actions 可以实行平台同步撰写，配置繁琐复杂，对 git 较熟悉。

hugo 也是一款不错的平台。推荐主题：paper

新手可以试试 Gridea。

进阶用法

魔改主题 + 又拍云存储（有相关上传插件，推荐）+ 备案域名

评论解决方案

valine + admin valine

稍微方便一点可以选动博。

VPS/自己的服务器

wordpress + 宝塔 + cdn + 备案域名

免费/付费空间

typecho + cdn +备案域名

主题推荐

免费 VOID

付费 cuteen 60

handsome 80

小提示#

注意做 DDOS与 CC 防护。推荐国外解析至免费 cloudfare cdn，教程稍后放。国内推荐阿里云。 静态资源可以试着用 jsdelivr白嫖，速度非常理想，静博同理。
